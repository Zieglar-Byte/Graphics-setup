Instruction for adding graphics file 
***************************************************************************************************

1) Copy "graphics.h" and "winbgim.h" files then paste into MinGW/include folder. 
   Default location is ("C:\MinGW\include\")

2) Copy "libbgi.a" to file then paste into MinGW/lib folder. 
   Default location is ("C:\MinGW\lib\")

3) Done...

NOTE:
1) The graphics.h will work only in the program saved in ".cpp" format.
